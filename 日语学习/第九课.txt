# 语法
## N は A1　です。
	日语形容词分为两类，一类形容词，二类形容词
	一类形容词以（い）结尾
	敬体时在后面加（です）
	
	例子：	四川料理は辛いです。		四川菜是辣的
			あのスープは　冷たいです。	那个汤很凉
			このお茶は　暑いです。		这个汤很烫
			
## A1做位于时的否定形式
	否定形式：
		（い） 变成 （く）+ （ないです）	
							（ありません）否定的尊敬形式
	
	熱い：熱くないです	-- 句末です可去掉，熱くない
	寒い：寒くないです
	高い：高くないです
	特殊词：いいです　变为	よくないです／よくありません
	
	例子：	このスープは熱くないです。	这个汤不烫
			今日は寒くないです。		今天不冷
			このカメラは高くありません		这个照相机不贵
			
##	一类形容词做谓语时的过去形式
	过去式敬体：（い）变成 （かった）+（です）
	过去的否定式：（い） 变成 （くなかったです）/（くありませんでした）
	
	例子：	旅行は　楽しかった	です。		旅行很有趣
			昨日は	寒くなかった　です。	昨天不冷
			昨日は	寒くありませんでした。	昨天不冷(尊敬)
			
	描述过去体验过的事物性质时候必须要使用过去式
	
##	一类形容词 + 名词
	一类形容词可直接修饰名词
	形容词和名词之间不能加（の）
	例：（美味しいの料理）这样是不对的，正确的是（美味しい料理）
	
##	を　变成　は
	（を）前面的名词做话题或者对比的时候，可以替换成（は）
	
##	あまり　A1/V(否定)
	（あまり）与后面的否定形式呼应，表示程度不难么高。
	不那么~
	例子：このスープは　あまり　熱くないです。 　这个汤不那么热
	
##	表示程度的副词
	とても／たいへん				很，非常
	すこし（少し）／ちょっと		一点儿
	あまり～ません					不太~
	ぜんぜん（全然）~　ません		根本不~
	
	あまり／全然 要与后面的否定形式呼应使用
	
##	语法总结
	形容词 + です											形容词敬体
		　 - い + く　+　ないです／くありません				形容词否定敬体
		　 - い + かった + です								形容词过去式
		   - い + く + なかったです／ありませんでした		形容词过去否定式
		   
	形容词 + 名词		直接修饰名词
	あまり + 形容词的否定形式	表示程度不难么高
	表示程度的副词
