语法部分
#　あります/います
	用来表达事物的存在的谓语，翻译为"有"
	"あります"存在的是非生命的物体
	"います"表示的是有生命的物体的存在，如人、动物等等

##	句型
###	句型一
	名词（场所）に 名词（物、人） が　あります／います　
	に 	提示地点
	が	提示主语
	あぃます	用于无意识物体
	います		用于有意识物体
	在 ~ 有 ~
	
	例子：	ここ　に　猫　が　います　。 在这里有猫 
			あそこ　に　だれ　が　いますが　。　在那里有谁吗？
	
###	句型二
	名词（物、人） は 名词（场所）に あります／います　
	は	提示主题
	に	提示地点
	~ 在 ~ 
	
	例子：	吉田さん　は　ここ　に　います。　　吉田桑在这里
			図書館（としょかん）　は　どこ　に　ありますか。　　图书馆在这里吗？

	语法变种
	N は　どこ　ですか	。 这个之前学习的语法和当前语法一致。

#	N と N 并列
	"と"连接两个或者两个以上的名词，表并列关系
	
	例子：ビール　と　ウイスキー	啤酒和威士忌
	
#	表示方位名词
	
	表示具体位置：名词 +　の + うえ（上）/した（下）/まえ（前）/うしろ（后）/となり（旁边）/なか（内）/そと（外）
	PS:日语的上和中文的上表达范围有区别，日语中只能表达上面，而在中文中的在墙壁上的表达不能使用 うえ 来表示。
	
	例子：	猫　は 箱　の　なか　に　います。	猫在箱子里面。
			機	の　うえ　に　猫　が　います。	在桌子上面有猫
			
#	ね	表示确认
	语气终助词"ね"接在句末
	表示就某事征求听话人同意
	
	例子：	あそこ　に　ねこ　が　ありますね。 在那里有猫欸

#	疑问词 + も + V(否定)
	此句型表示全面否定
	
	例子： （冷蔵庫）れいぞうこ　に　何　も　ありますん。 冰箱里什么都没有
			教室（きょうしつ）　に　だれ　も　いますん。	教室里一个人也没有
