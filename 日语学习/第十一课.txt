#	语法

##  N1　は　N2　が　A1/A2　です
    A1/A2为表示情感的形容词
    N1表示情感的主体(主语)，N2表示情感的对象(谓语)
    
    例子：  小野さんは歌が好きです　小野喜欢唱歌
            森さんは肉が嫌いです    森讨厌肉

##  N1　は　N2　が　分かります・できます
    わかります  知道，懂
    できます    做到
    上手です    擅长
    下手です    不擅长
    苦手です    不擅长
    等有关能力的词
    N1表示能力的主体(主语)，N2表示能力的对象(谓语)

    例子：  スミスさんは日本語が分かります　斯密斯懂日语
            私はコンピュータができません　  我不会用电脑
            小野さんは料理が苦手です　小野不擅长做饭

##  N や N
    相当于汉语的“~呀~什么的”
    常用搭配：~ や ~ など
    や 用于只列举许多项目中的一个
    と 用于列举所有的项目

    例子：   森さんは時々中国やアメリカへ行きます   森先生有时去中国或美国
            小野さんはサッカーやテニスや ゴルフ	が　すきです。    小野先生喜欢足球、高尔夫球等等      

##  ~ から 和 だから （原因/理由）
	助词：紧跟着前面的句子
	连词：连结两个词或者两个句子的词，前后有标点符号
    （から）助词，表原因
    例：もりさんはお酒がすきですから、毎日飲みます。	森先生喜欢酒，所以每天喝
		寒いですから、窓を閉めます。					因为天气冷，所以关窗
        
    （だから）连词，表原因，理由
	例： キムさんは野菜が嫌いです。だから、あまり野菜を食べません。　金先生讨厌蔬菜，所以，不怎么吃蔬菜
		
##	表示频率的副词
	いつも			总是，经常
	よく            常常
	時々            有时
	たまに          偶尔
	あまり～ない    不怎么
	全然～ない      完全不
	
	例子：	私はたまに　コンサート　へ　いきます。	我偶尔去演唱会
		　	小野さんは時々映画を見ます。	小野时不时去看电影
		
##	*どうしてですか
	询问事由时使用，
	相当于汉语的“为什么”，情感比较强烈，比如青梅竹马被横插，十年感情不如一周，为什么啊
	回答时在谓语后面加（から）
